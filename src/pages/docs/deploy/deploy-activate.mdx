## 初始化向导

配置以下环境变量，启用初始化向导

```
STEEDOS_TENANT_ENABLE_ACTIVATION=true
```

如果是全新的数据库，初始化向导会执行以下操作：

* 在本地数据库中创建企业。
* 创建管理员账户，管理员用户名为手机号，初始密码为华炎云管理控制台中配置的默认密码。
* 自动下载安装当前企业已购买的软件包。
* 自动创建API_KEY，并配置开发环境所需的 metadata 相关环境变量

## 注册账户

请进入华炎魔方客户门户（<https://console.steedos.cn/>），并登录进系统。如果没有账户，请先注册。

### 新建我的魔方

在管理控制台 - 我的魔方中，新建我的魔方。

在这里设置的初始密码就是激活过程中自动为本地魔方项目创建的初始管理员账户的默认密码。

### 新建私有部署

选择我的魔方，在详情页，点击私有部署处的“新建”按钮，即可新建一条私有部署记录。

创建的私有部署记录中，包含魔方ID和 API Key，需在后续步骤中，配置为私有部署中的环境变量。

* 魔方ID；
* 私有部署的API Key 。

### 初始化华炎魔方

点击私有部署记录详细界面右上角的`复制激活参数`按钮可以把相关环境变量配置复制下来。

粘贴到华炎魔方系统中即可完成激活流程。
