华炎魔方基于 [Nodejs](https://nodejs.org/zh-cn/) 开发，运行环境需要 [mongodb 数据库](https://www.mongodb.com/docs/v4.2/administration/install-community/) 和 [redis 缓存](https://redis.io/docs/getting-started/) 服务。


使用以下命令，可以在本地创建华炎魔方模版项目。

```shell
npx create-steedos-app my-app
```

## Docker部署

如果您只是想试用华炎魔方的功能，建议使用 Docker 部署。

- [docker-compose.yml](https://gitlab.steedos.cn/steedos/docker/-/tree/main/steedos-community)
- [Docker 私有部署说明](./deploy-docker)
